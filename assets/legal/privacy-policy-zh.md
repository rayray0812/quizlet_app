# 隱私權政策 — 拾憶（Recall）

**最後更新日期：2026-04-21**
**聯絡信箱：ruiruiclips@gmail.com**

拾憶（以下稱「本應用程式」）尊重並保護每一位使用者的個人隱私。本政策說明我們如何蒐集、使用、儲存及保護您的個人資料。

## 1. 資料蒐集範圍

### 1.1 訪客模式（不登入）
- **不蒐集任何個人識別資料**
- 所有學習集、複習紀錄、設定皆儲存於您的裝置（Hive 本地加密資料庫）
- 不會上傳任何資料到伺服器

### 1.2 登入模式（選用）
我們使用 Supabase 作為雲端同步服務，若您選擇登入，將蒐集：
- **帳號資訊**：電子郵件、加密密碼（或 OAuth 提供者 ID）
- **學習內容**：學習集、卡片、複習進度、複習紀錄
- **裝置資訊**：建立/更新時間戳記（用於同步衝突解決）

## 2. 第三方服務

本應用程式整合以下第三方服務：

| 服務 | 用途 | 資料範圍 |
|------|------|----------|
| Supabase | 帳號驗證 + 雲端同步 | 僅於登入模式啟用 |
| Google Gemini API | 拍照建卡 AI 辨識（選用） | 使用者自行提供 API Key，圖片直接傳送至 Google，本應用程式不保存 |
| Groq Cloud | 拍照建卡 AI 辨識（選用） | 使用者自行提供 API Key，圖片直接傳送至 Groq，本應用程式不保存 |
| Unsplash | 卡片圖片搜尋（選用） | 僅搜尋查詢字串 |

**重要：AI API Key 僅儲存於您的裝置（FlutterSecureStorage），不會上傳到本應用程式的伺服器。**

## 3. 裝置權限

| 權限 | 用途 | 是否必要 |
|------|------|----------|
| 相機 | 拍照建卡（F6） | 否（選用功能） |
| 相簿讀取 | 從相簿選圖建卡 | 否 |
| 通知 | 每日複習提醒 | 否（使用者可關閉） |
| 生物辨識 | 快速解鎖 | 否 |
| 網路 | 雲端同步、AI 辨識 | 登入/AI 功能所需 |

## 4. 資料保存期限

- **訪客模式**：資料永久保存於您的裝置，直到您自行刪除
- **登入模式**：資料保存於 Supabase，直到您刪除帳號

## 5. 您的權利

您隨時可以：
- **匯出資料**：於「設定 → 帳號與安全」匯出 JSON 備份
- **刪除帳號**：於「設定 → 帳號與安全 → 刪除帳號」即時刪除所有雲端資料
- **關閉同步**：登出即停止雲端同步，本地資料保留

## 6. 兒少保護

本應用程式主要服務對象為高中學生。我們：
- 不要求使用者提供真實姓名、地址、電話
- 不收集精準位置資訊
- 不在應用程式內投放廣告
- 建議 13 歲以下使用者需在家長同意下使用

## 7. 資料安全

- 本地資料：Hive + AES 加密（FlutterSecureStorage 管理金鑰）
- 雲端傳輸：HTTPS/TLS 1.2+
- 密碼：Supabase 採 bcrypt 雜湊，永不以明文儲存

## 8. 政策修改

若本政策有重大變更，我們將於應用程式內通知您。
