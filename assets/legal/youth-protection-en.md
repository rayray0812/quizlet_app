# Youth Protection Notice — Recall

**Last updated: 2026-04-21**

## Design Principles

Recall is primarily designed for high-school students (ages 15–18). We apply the following youth-protection measures:

### 1. Minimal Data Collection
- Sign-up requires only email (no real name, birthday, phone, or address).
- No precise location data.
- No advertising ID tracking.
- Guest mode is fully offline — no server records are created.

### 2. No Advertising
- No ads of any kind in the App.
- No third-party ad SDKs.
- User data is never sold.

### 3. Paid Features
- All core features are free.
- Photo-to-flashcard requires a user-provided free API key (Gemini / Groq). Costs are transparent and controlled by the user.

### 4. Social Features Limits
- Public study-set sharing is opt-in; private by default.
- Share codes authorize only the recipient; no public index.
- No direct messaging or user-to-user interaction.

### 5. Parental Guidance
- Users under 13 should use the App with parental assistance.
- Parents can delete all data at any time via Settings → Account & Security → Delete Account.
- We recommend parents and children agree on daily review time to avoid overuse.

## App Store Age Rating

- **iOS App Store**: 4+ (no objectionable content)
- **Google Play**: Everyone

## Reporting Inappropriate Content

If you find inappropriate content (violence, sexual content, hate speech) in a shared study set, email ruiruiclips@gmail.com. We respond within 72 hours.
