# 兒少保護聲明 — 拾憶（Recall）

**最後更新日期：2026-04-21**

## 設計原則

拾憶主要服務對象為高中學生（15–18 歲）。我們在設計上採取以下兒少保護措施：

### 1. 最小資料蒐集
- 註冊僅需 email（不要求真實姓名、生日、電話、地址）
- 不蒐集精準位置資訊
- 不使用裝置廣告 ID 進行追蹤
- 訪客模式下完全離線可用，不產生任何伺服器紀錄

### 2. 無廣告
- 應用程式內不投放任何形式的廣告
- 不嵌入第三方廣告 SDK
- 不販售使用者資料

### 3. 付費功能
- 所有核心功能免費
- AI 拍照建卡需使用者自行申請免費的 API Key（Gemini/Groq），費用透明且使用者自行控管

### 4. 社群功能限制
- 公開學習集功能需手動開啟，預設為私人
- 分享碼僅授權給接收者，無公開索引
- 不提供即時訊息或使用者互動功能

### 5. 家長建議
- 13 歲以下使用者建議在家長協助下使用
- 家長可隨時於「設定 → 帳號與安全 → 刪除帳號」移除所有資料
- 建議家長與孩子共同設定每日複習時間，避免過度使用
